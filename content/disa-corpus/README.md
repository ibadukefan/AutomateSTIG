# DISA Corpus

This directory tracks authoritative DISA Cyber Exchange artifacts and generated metadata used to pursue every-DISA-STIG automation.

- `download-index.json` — normalized public DISA download index.
- `artifacts.manifest.json` — authorized artifact metadata with source URLs and SHA-256 hashes.
- `status.json` — current high-level corpus/automation counts.

Large or redistribution-sensitive artifacts may be tracked in metadata-only mode instead of committed directly.
